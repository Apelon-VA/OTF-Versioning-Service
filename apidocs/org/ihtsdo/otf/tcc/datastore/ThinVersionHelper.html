Class ThinVersionHelper

  • java.lang.Object
    • org.ihtsdo.otf.tcc.datastore.ThinVersionHelper

  • public class ThinVersionHelper
extends Object
    Granularity of SNOMED version stamp is seconds, not milliseconds. Version stamps to not need to go before 1960 since SNOMED was created in the 1960s. Integer.MAX_INT = 2,147,483,647 If this is seconds, then it represents 35,791,394 minutes 596,523 hours 24,855 days 68 years So if we set 0 = Jan 1, 2028 (1830407753464L), and granularity = seconds, we can represent versions from 1960 -> 2096.
