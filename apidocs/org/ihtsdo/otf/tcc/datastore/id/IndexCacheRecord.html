org.ihtsdo.otf.tcc.datastore.id

Class IndexCacheRecord

  • java.lang.Object
    • org.ihtsdo.otf.tcc.datastore.id.IndexCacheRecord

  • public class IndexCacheRecord
extends Object
    Stores cross-reference information for origin relationships, destination relationship origins, and refex referenced components in a integer array, minimizing the object allocation burden that would otherwise be associated with this information. This class interprets and manages the contents of that array.

    Implementation notes

    See the class RelationshipIndexRecord for documentation of the structure of the relationship index data.
