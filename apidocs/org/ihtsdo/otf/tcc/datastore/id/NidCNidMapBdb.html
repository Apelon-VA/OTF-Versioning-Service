extends ComponentBdb

    Implementation Details

    The nid is the (nid - Integer.MIN_VALUE) index into an int[] which stores the cNid.

    This single array approach is taken because Java stores multidimensional arrays as arrays of arrays, rather than a contiguous block of arrays. Each array has an overhead of 96 bits (above and beyond its data), which doubles the memory size, and also increases the burden on the garbage collector.
