Class BdbPathManager

  • java.lang.Object
    • org.ihtsdo.otf.tcc.datastore.BdbPathManager

  • public class BdbPathManager
extends Object
    OldPath management. Defines methods for obtaining and modifying paths. Paths are now stored/defined in reference sets (extension by reference). This implementation avoids the use of the redundant OldPath store and instead marshals to to the Extension store (indirectly).
