Class IndexedGenerationCallable

    public class IndexedGenerationCallable
extends Object
implements Callable<Long>
    A Callable<Long> object that will block until the indexer has added the document to the index. The call() method on the object will return the index generation that contains the document, which can be used in search calls to make sure that generation is available to the searcher.
