    public class TtkRefexArrayOfByteArrayRevision
extends TtkRevision
    The Class TkRefexArrayOfByteArrayRevision represents a version of an array of byte array type refex member in the eConcept format and contains methods specific for interacting with this version. Further discussion of the eConcept format can be found on TkConcept.
