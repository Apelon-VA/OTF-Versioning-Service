org.ihtsdo.otf.tcc.api.lang

Enum LanguageCode

  • java.lang.Object
    • java.lang.Enum<LanguageCode>
      • org.ihtsdo.otf.tcc.api.lang.LanguageCode
  • All Implemented Interfaces:
    Serializable, Comparable<LanguageCode>

    public enum LanguageCode
extends Enum<LanguageCode>
    The Enum LanguageCode represents a two or six character representation of the language or language and dialect code respectively. If the language has a dialect, the dialect is represented by a dash and two characters following the language.
    See Also:
    http://en.wikipedia.org/wiki/List_of_ISO_639-1_codes
