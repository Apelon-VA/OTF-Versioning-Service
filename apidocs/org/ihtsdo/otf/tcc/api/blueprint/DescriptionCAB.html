    PropertyChangeListener, EventListener

    public class DescriptionCAB
extends CreateOrAmendBlueprint
    The Class DescriptionCAB contains methods for creating a description blueprint. This blueprint can be constructed into a type of DescriptionChronicleBI. This is the preferred method for updating or creating new descriptions.
    See Also:
    TerminologyBuilderBI, DescriptionChronicleBI
